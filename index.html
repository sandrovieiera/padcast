podcast 1

O que é Front-End? Front-end é a parte visual e interativa de um site ou aplicação web com a qual os
usuários interagem diretamente. Ele engloba o design, a estrutura, e o
comportamento das páginas da web. Utiliza tecnologias como HTML, CSS e JavaScript
para criar interfaces amigáveis e funcionais. O objetivo do front-end é garantir
que o usuário tenha uma experiência agradável e intuitiva ao navegar pelo site.
Desde a disposição dos elementos na página até a execução de animações e
interações dinâmicas, tudo faz parte do trabalho de um desenvolvedor front-end.
Quer saber mais sobre front-end? Então, ouça nossos podcasts logo abaixo!

ex:

A classe .card é aplicada a um div que será estilizado.
width e height definem as dimensões do cartão.
background-color define a cor de fundo do cartão.
border e border-radius definem a borda e os cantos arredondados do cartão.
box-shadow cria a sombra projetada: 10px 10px 5px rgba(0, 0, 0, 0.5) significa uma
sombra deslocada 10 pixels para a direita e 10 pixels para baixo, com um desfoque de 5 pixels e uma cor preta com 50% de opacidade.
display, align-items, justify-content, e font-family são usados para centralizar o
texto dentro do cartão e definir a fonte.
margin é usada para centralizar o cartão na página.

ex:

O evento window.onload garante que o código dentro dele será executado apenas depois
que toda a página tiver sido carregada.
setTimeout(function() { alert('Bem-vindo à Força do Front-End!'); }, 3000); cria um
temporizador que executará a função de alerta após 3000 milissegundos (ou 3 segundos).
Quando você abrir esse código em um navegador, um alerta com a mensagem "Bem-vindo à
Força do Front-End!" aparecerá 3 segundos após o carregamento da página.
